Smart Attendance System (QR-Based)

A secure and intelligent web-based attendance system designed to eliminate proxy attendance and automate the entire attendance process. This system uses a dynamic QR code mechanism combined with real-time location validation to ensure that only students who are physically present in the classroom can mark their attendance. The application is built using modern web technologies and deployed online for easy access.

The system generates a unique QR code that refreshes every 10 seconds, making it impossible for students to reuse or share the code. When a student scans the QR code, the system verifies their real-time location and records attendance instantly with accurate timestamps. All data is securely stored in a MongoDB database and can be accessed by administrators for monitoring and analysis.

What this project demonstrates:

  • Anti-proxy mechanism: Implementation of a dynamic QR code that refreshes every 10 seconds to prevent unauthorized attendance marking
  • Location-based validation: Ensures that attendance is marked only when the student is physically present in the required location
  • Real-time data handling: Instant attendance marking with accurate timestamp and seamless data flow between frontend and database
  • Full-stack development: Integration of frontend (React.js) with backend database (MongoDB) for efficient data management
  • Deployment & accessibility: Hosted using Render, making the application accessible from anywhere without local setup
  • User-friendly interface: Clean and responsive UI designed for ease of use by both students and administrators
